# Python Quiz Application V9

A Python-based quiz application with multiple difficulty levels, timer, leaderboard, and score tracking.

## Features

* Easy, Medium and Hard difficulty levels
* Random question order
* Case-insensitive answer checking
* 10-second timer for each question
* Displays the correct answer when the user answers incorrectly
* Percentage calculation
* Performance feedback messages
* Score saving using file handling
* Leaderboard system

## Technologies Used

* Python
* Functions
* Loops
* Lists
* File Handling
* Random Module
* Time Module

## How to Run

1. Download the project.
2. Open the file in IDLE or any Python editor.
3. Run `quiz_app_v9.py`.
4. Enter your name and start playing.
